# GuideVisualizer 2.0

A web-based visualization platform for exploring and comparing Cas13 guide RNA predictions generated by multiple guide design tools.

## Overview

GuideVisualizer 2.0 is an interactive Flask-based web application developed to visualize guide RNA predictions across multiple Cas13 guide design tools for *Plasmodium falciparum* transcripts.

The application allows users to:

- Visualize guide RNA positions on transcripts
- Compare predictions from multiple Cas13 guide design tools
- Display CDS, 5' UTR, and 3' UTR regions
- View guide scores from each prediction tool
- Filter Top-N guides independently for each tool
- Explore transcript-level guide distributions interactively

## Installation

### Clone the repository

```bash
git clone https://github.com/<your_username>/GuideVisualizer2.git

cd GuideVisualizer2
```

---

## Build Docker Image

```bash
docker build -t guidevisualizer:2.0 .
```

---

## Run GuideVisualizer

```bash
docker run --rm \
-p 5001:5001 \
-v /path/to/Guideprediction:/data \
-e GUIDEVIS_DATA=/data \
guidevisualizer:2.0
```

Open your browser:

```
http://localhost:5001
```
