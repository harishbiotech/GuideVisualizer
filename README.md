# GuideVisualizer 2.0

A Dockerized web application for interactive visualization and comparison of Cas13 guide RNA predictions generated by multiple guide design tools for *Plasmodium falciparum*.

---

## Overview

GuideVisualizer 2.0 is a Flask-based visualization platform developed to explore, compare, and analyze guide RNA predictions generated by multiple Cas13 guide design tools.

The application enables users to:

- Visualize guide RNA positions on transcript sequences
- Compare predictions from multiple Cas13 guide design tools
- Display CDS, 5' UTR, and 3' UTR regions
- View guide scores generated by each prediction tool
- Filter Top-N guides independently for each tool
- Explore transcript-level guide distributions interactively

## Installation

### Clone the repository

```bash
git clone https://github.com/harishbiotech/GuideVisualizer.git

cd GuideVisualizer
```

---

## Build Docker Image

```bash
docker build -t guidevisualizer:2.0 .
```

---

## Run the Application

```bash
docker run --rm \
-p 5001:5001 \
-v /path/to/Guideprediction:/data \
-e GUIDEVIS_DATA=/data \
guidevisualizer:2.0
```

Open your browser:

```
http://localhost:5001
```

## Dataset

The guide prediction dataset is distributed separately through Zenodo.

**Zenodo Dataset**

https://zenodo.org/records/21337342

The dataset contains:

- Guide RNA predictions generated by TIGER
- Guide RNA predictions generated by DeepCas13
- Guide RNA predictions generated by CASOwary
- Guide RNA predictions generated by Cas13design
- Transcript FASTA sequences
- CDS FASTA sequences

After downloading, organize the dataset as:

```
Guideprediction/
├── input/
└── Results/
    ├── Cas13_result/
    ├── CASowary_result/
    ├── DeepCas13_result/
    └── tiger_result/
```
